# snake game 贪吃蛇
游戏介绍：复刻经典贪吃蛇小游戏
- 随机食物与随机 snake 开始位置
- 固定网格大小
- 方向键可操控蛇的移动
- 按键与蛇运动方向相同会让蛇以 2 倍速度移动
- 碰撞墙壁和蛇身检测
- 吃到食物蛇神增长 1 格
